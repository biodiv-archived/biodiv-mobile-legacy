Version : 1.0.0

Login

You can log in to the portal from this page. If you dont have an account, you can register as a new user from New User button. If you forgot the password you can get it through Forgot Password button


Home

After you have succesfully logged in, you will be redirected to home page.


New Observation

1.Click the "Add Photos" button to add images from the gallery or take new images with the camera.

2. Location and date will be automatically set to the last known place and time as provided by the device. To edit location, click the GPS button and get updated location or click edit location to pinpoint it on a map. Date can be corrected from the date picker.

3.If you know the scientific name and common name of the observation, you can enter it or else click on "Help Identify" button

4.Select the user groups to which the current observation should be posted to.

5.Click submit. The observation will be uploaded instantly if an internet connection is available. If not it will be queued until a connection is available.


Browse Observations

Click the Browse Observation button and select the species group to bring up the list of latest observations. Observation details can be seen by clicking each observation. Swipe to move to next observation or to go back to previous.


Observations near me

Using this, observations known to exist in nearby locality can be retrieved. Note: If there are no observations on the portal in the immediate region, it will fetch observations that are closest to the current location.


My Observations

This will retrieve and display all your uploaded observations.

Joined Groups

Use this to discover and join all available groups on the portal. Only groups that you join will be available for posting observations during upload.


Observation Queue

Observations pending upload or have failed to upload will be displayed here with its upload status. Observations currently being uploaded will show the "processing" status.


Settings

Upload over Wifi only: Switch this on to upload only when a wifi connection is enabled.

Upload manually only: Prevents automated uploading. Each pending observation will need to be manually enabled for upload.
